# WeTouch

一个极其简洁的记账App, UI简单清爽，支持基本记账功能



## 主要特性

- 消费分类完全自定义，并且支持删除功能，完全取决于你的需要；
- 可按照年/月/日/标签分类检索你的账单，并提供图表式分析，使你的消费记录一目了然。
- 支持记录一键导出账单功能，保证你的流水账单不丢失。
